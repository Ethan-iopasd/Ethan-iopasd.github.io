pandas的常用方法

Pandas 常用方法整理

1. 数据读取与写入

pd.read_csv(file_path, sep=’,’, header=True) # 读取 CSV 文件
pd.read_excel(file_path, sheet_name=0) # 读取 Excel 文件
pd.read_sql(query, connection) # 从 SQL 数据库读取数据
df.to_csv(file_path, index=False) # 写入 CSV 文件
df.to_excel(file_path) # 写入 Excel 文件

2. 数据结构创建

pd.DataFrame(data, columns=[…], index=[…]) # 创建 DataFrame
pd.Series(data, index=[…]) # 创建 Series

3. 数据查看与检查

df.head(n=5) # 显示前 5 行
df.tail(n=5) # 显示后 5 行
df.info() # 显示数据类型、内存等基本信息
df.describe() # 显示各列统计指标（均值、标准差等）
df.shape # 返回 (行数, 列数)
df.dtypes # 显示各列数据类型

4. 数据选择与过滤

df[‘column_name’] # 选择单列
df.loc[row_label, col_label] # 按标签选择数据（行列名）
df.iloc[row_idx, col_idx] # 按位置索引选择数据（行列序号）
df.filter(condition) # 按条件过滤行（如 df[df[‘age’] > 20]）

5. 数据处理与清洗

df.fillna(value=0, method=’mean’) # 填充缺失值
df.dropna(axis=0) # 删除缺失值所在的行/列（axis=0 行，axis=1 列）
df.drop_duplicates() # 删除重复行
df.replace(old_value, new_value) # 替换特定值
df.astype(new_dtype) # 转换列数据类型（如 int→float）

6. 数据修改操作

df.set_index(‘column’) # 设置新索引列
df.rename(columns={‘old’:’new’}) # 重命名列
df.sort_values(by=’column’, ascending=True) # 按列排序
df[‘new_col’] = values # 添加新列（直接赋值）
df.drop(columns=’column’, axis=1) # 删除列（axis=1 表示列）

7. 统计分析

df.mean() # 各列均值
df.median() # 各列中位数
df.sum() # 各列总和
df.max() / df.min() # 各列最大值/最小值
df.std() # 各列标准差
df.corr() # 列间相关系数矩阵
df.cov() # 列间协方差矩阵

8. 数据合并与连接

pd.concat([df1, df2], axis=0) # 沿行（纵向）或列（横向）合并
pd.merge(df1, df2, on=’key’, how=’inner’) # 按键合并（inner/outer/left/right join）

9. 分组与聚合

grouped = df.groupby(‘group_column’) # 按列分组
grouped.count() # 分组计数
grouped.sum() # 分组求和
grouped.agg({‘col’: ‘max’}) # 自定义聚合函数（如多列不同聚合方式）

10. 时间序列处理

pd.to_datetime(df[‘date_col’]) # 转换为日期格式
df.resample(‘时间频率’).mean() # 按时间频率重采样（如 D→按天）

11. 数据可视化（依赖 matplotlib）

df.plot(kind=’line’, x=’col1’, y=’col2’) # 生成线图/柱状图/饼图等基础图表
