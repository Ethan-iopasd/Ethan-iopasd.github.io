vim常用命令

Vim 常用命令指南

模式切换

命令 描述
i 进入插入模式（当前光标前）
a 进入插入模式（当前光标后）
o 在当前行下方新建插入行
O 在当前行上方新建插入行
v 进入可视模式（字符选择）
V 进入可视模式（行选择）
Esc 返回普通模式

光标移动

基础移动

命令 描述
h 左移
j 下移
k 上移
l 右移
w 跳到下一个单词开头
b 跳到上一个单词开头

快速跳转

命令 描述
gg 跳转到文件开头
G 跳转到文件末尾
:n 跳转到第n行（n为数字）
0 跳转到行首
$ 跳转到行尾
Ctrl+f 向下翻页
Ctrl+b 向上翻页

文本编辑

删除操作

命令 描述
x 删除当前字符
dw 删除到单词末尾
dd 删除整行
D 删除到行尾

复制粘贴

命令 描述
yy 复制当前行
p 粘贴到光标后
P 粘贴到光标前
yiw 复制当前单词

撤销/重做

命令 描述
u 撤销操作
Ctrl+r 重做操作

搜索替换

命令 描述
/keyword 向下搜索关键字
?keyword 向上搜索关键字
n 跳转到下一个匹配项
N 跳转到上一个匹配项
:%s/old/new/g 全局替换（加c确认每个替换）

文件操作

命令 描述
:w 保存文件
:q 退出
:wq / :x 保存并退出
:q! 强制退出不保存
:e filename 打开新文件
:saveas filename 另存为

分屏操作

命令 描述
:sp / :split 水平分屏
:vsp 垂直分屏
Ctrl+w hjkl 切换分屏窗口
Ctrl+w c 关闭当前分屏
Ctrl+w o 关闭其他分屏

其他实用技巧

  1. 显示行号：:set number（取消用:set nonumber）
  2. 批量缩进：可视模式选中文本后按>或<
  3. 重复命令：数字 + 命令（如3dd删除3行）
  4. 执行外部命令：:!command（如:!ls）

提示：Vim 的强大之处在于命令组合，例如 d2w = 删除2个单词，ci" = 修改引号内的内容
